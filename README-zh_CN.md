# Harpia: 自建 Kubernetes 集群

Harpia 是一套用于自建 Kubernetes 集群的工具和配置，组件包括集群部署，监控，网络和存储系统。
我们针对中小型集群进行了优化，希望能帮助用户组建自己的集群。

"Harpia" 即[角雕](https://en.wikipedia.org/wiki/Harpy_eagle)，是一个鹰的亚种，
热带雨林中最强大的掠食鸟类 。
